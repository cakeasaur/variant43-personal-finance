## Заключение

В ходе работы разработано кроссплатформенное приложение для управления личными финансами
(вариант 43) на базе Python + Kivy + SQLite.

### Реализованный функционал (v0.1)

- **Транзакции** — добавление доходов и расходов, фильтрация по типу и периоду,
  отображение итогов (баланс, суммы доходов/расходов);
- **Отчёты и визуализация** — агрегация расходов по категориям и по дням,
  диаграммы в UI;
- **Финансовые цели** — CRUD целей, атомарное пополнение (`deposit()`),
  отображение прогресса в процентах и абсолютных значениях;
- **Напоминания** — CRUD напоминаний с периодичностью (none/daily/weekly/monthly),
  пересчёт следующей даты при отметке выполнения, фильтр ближайших (`list_upcoming()`);
- **Защита данных** — шифрование локальной БД (AES-256-GCM + scrypt),
  без WAL-утечки данных во временной директории.

### Качество и инфраструктура

- 30+ автоматических тестов покрывают доменную логику, репозитории и криптографию;
- CI на GitHub Actions: линтер ruff + pytest + pip-audit (проверка CVE);
- код структурирован по слоям: `core` / `infra` / `ui` без циклических зависимостей.
