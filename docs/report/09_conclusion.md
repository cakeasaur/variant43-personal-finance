## Заключение

В ходе работы разработано кроссплатформенное приложение для управления личными финансами
(вариант 43) на базе Python + Kivy + SQLite.

Реализованы основные функции:
- учёт доходов/расходов и категоризация транзакций;
- отчёты и визуализация (диаграммы по категориям и по дням);
- финансовые цели и отображение прогресса;
- напоминания о платежах с периодичностью;
- защита данных путём шифрования локальной БД (пароль пользователя, scrypt + AES-GCM).

Также реализованы тесты и автоматические проверки качества/безопасности, что повышает надёжность
и воспроизводимость результата.
