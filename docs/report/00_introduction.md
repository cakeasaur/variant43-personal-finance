## Введение

### Актуальность
Управление личными финансами требует регулярного учёта доходов и расходов, анализа структуры трат и
контроля будущих обязательных платежей. Без инструмента учёта пользователь теряет прозрачность бюджета,
что усложняет планирование и достижение финансовых целей.

### Цель работы
Разработать кроссплатформенное приложение для управления личными финансами (вариант 43), обеспечивающее:
- учёт доходов/расходов;
- категоризацию транзакций;
- визуализацию данных (диаграммы);
- финансовые цели;
- напоминания о платежах;
- защиту данных (шифрование локальной БД).

### Задачи
- выполнить анализ предметной области и сформулировать требования;
- спроектировать архитектуру приложения и модель данных SQLite;
- реализовать приложение на Python + Kivy + SQLite;
- реализовать отчёты, цели, напоминания;
- реализовать защиту данных с локальным шифрованием;
- подготовить тесты и проверки качества;
- оформить пояснительную записку по структуре методички.

### Объект и предмет исследования
- **Объект**: персональные финансы пользователя (транзакции, цели, обязательные платежи).
- **Предмет**: методы и технологии разработки кроссплатформенного приложения с локальным хранением и защитой данных.
