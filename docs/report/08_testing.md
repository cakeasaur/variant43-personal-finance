## 5 Тестирование

### 5.1 Виды тестов
- **Unit-тесты** доменной логики: агрегаты (итоги, расходы по категориям/дням), валидации моделей.
- **Интеграционные тесты** слоя SQLite: CRUD, транзакционность, выборки и порядок сортировки, цели/напоминания.
- **Unit-тесты крипто-модуля**: корректность encrypt/decrypt и обработка неверного пароля.

### 5.2 Инструменты
- `pytest` — запуск тестов;
- `ruff` — статический анализ;
- `pip-audit` — проверка зависимостей на известные уязвимости;
- GitHub Actions — автоматический прогон проверок на `main/develop` и PR.

### 5.3 Команды запуска
Из корня проекта:
```powershell
py -3.12 -m ruff check .
py -3.12 -m pytest -q
py -3.12 -m pip_audit -r requirements/app.txt -r requirements/dev.txt
```
